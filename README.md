# KindaVM

USB HID keyboard and mouse emulation for KVM usage on Raspberry Pi.

## Hardware Requirements

- Raspberry Pi Zero 2W
- Geekworm X630 HDMI to CSI-2 capture card
- Micro USB to USB (A or C) cable for HID (keyboard/mouse) device
- Micro USB to USB (A or C) cable for RPI power (optional - power can be drawn from HID cable)
- HDMI cable for video capturing

## Installation

> [!IMPORTANT]
> `sudo` is required

```bash
curl -sSL https://github.com/Ch00k/kindavm/releases/latest/download/install.sh | bash
```
